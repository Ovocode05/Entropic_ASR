# Entropic-ASR  
### Solving the “Phonetic Collision” Problem in Hinglish Financial Voice Transactions
## Vision

Most voice assistants treat Hinglish (Hindi + English) as a broken variant of a single language. In reality, it is a **high-entropy bilingual signal** where languages switch mid-sentence.

**Entropic-ASR** is a domain-specialized speech recognition engine designed for financial voice transactions in emerging markets. It goes beyond transcription by combining neural and symbolic reasoning to resolve phonetic ambiguities in real-world UPI-style interactions.

This project focuses on reliability, latency, and robustness in noisy environments.
